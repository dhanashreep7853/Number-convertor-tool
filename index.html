📚 Number Conversion Notes:
[A=10, B=11, C=12, D=13, E=14, F=15]
[Base: Binary = 2(0,1), Octal = 8(0-7), Decimal = 10(0-9), Hexadecimal = 16(0-9,A-F)]
  • Binary → Decimal: Multiply digits by 2ⁿ, then sum.
    Ex: 1001 → 1×2³+0×2²+0×2¹+1×2⁰ = 9
  • Binary → Octal: Group binary in 3 bits from right.
    Ex: 10101 → 010 101 → 25
  • Binary → Hex: Group in 4 bits from right.
    Ex: 110101011 → 0001 1010 1011 → 1AB
  • Decimal → Binary: Divide by 2, reverse remainders.
    Ex: 13 = 1101

  • Decimal → Octal: Divide by 8, reverse remainders.
    Ex: 78 = 116
  • Decimal → Hex: Divide by 16, reverse remainders.
    Ex: 350 = 15E
  • Octal → Binary: Convert each digit to 3-bit.
    Ex: 247 → 010 100 111 = 10100111
  • Octal → Decimal: Multiply digits by 8ⁿ, then sum.
    Ex: 177 = 127
  • Octal → Hex: Convert to Binary → Hex.
    Ex: 5456 → 101100101110 = B2E
  • Hex → Binary: Each hex digit to 4-bit.
    Ex: ACD2 → 1010 1100 1101 0010 = 110011010010
  • Hex → Decimal: Multiply digits by 16ⁿ, sum.
    Ex: 3A4 = 932
  • Hex → Octal: Hex → Binary → Octal.
    Ex: B28 → 101100101000 = 5450
